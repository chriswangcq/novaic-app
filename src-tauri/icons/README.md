# NovAIC 图标资源

## 应用图标（Dock / 启动台）

- **macOS 规范**：使用 1024×1024 画布，**不要**自己加圆角，系统会自动应用 squircle。
- **安全区**：主内容放在约 90% 区域内（例如 1024px 下内容区约 920×920），留出边距，视觉上会更小、更圆、更统一。
- **格式**：`icon.icns` 需包含多尺寸（16、32、128、256、512、1024 等），可用 `iconutil` 或 Tauri/Electron 常用工具从 PNG 生成。

## 托盘图标（菜单栏）

- **tray-icon.png**：用于菜单栏，在 macOS 上会以 **template** 方式显示为黑白。
- **设计**：单色（黑或白）图形 + 透明底，不要渐变或彩色，系统会按菜单栏深浅自动反色。
- 若使用彩色图，设为 template 后仍会按透明度做遮罩，但效果可能不理想，建议用纯黑白图。
